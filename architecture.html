Security & Compliance

Defense in Depth

Layered controls for a financial-grade application.

🔐 Security Controls
  • TLS 1.3 in transit · AES-256 at rest
  • Row-Level Security on all 14 tables
  • JWT auth with short expiry + refresh
  • Server-only secrets (service role, AI keys)
  • Admin client isolated to server routes
  • File validation — MIME whitelist, 20MB cap
  • Private storage bucket, per-user folders
  • No card/credential entry permitted by design
